Теоретичні відомості

Алгебра

Формули скороченого множення

  • a² − b² = (a − b)(a + b)
  • (a + b)² = a² + 2ab + b²
  • (a − b)² = a² − 2ab + b²

Квадратне рівняння

ax² + bx + c = 0,  a ≠ 0

D = b² − 4ac

x₁,₂ = (−b ∓ √D) / (2a)

Модуль числа

|a| = { a, якщо a ≥ 0; −a, якщо a < 0 }

Степені

  • aᵐ·aⁿ = aᵐ⁺ⁿ
  • aᵐ / aⁿ = aᵐ⁻ⁿ
  • (aᵐ)ⁿ = aᵐⁿ
  • (ab)ⁿ = aⁿ bⁿ
  • a⁻ⁿ = 1/aⁿ (a ≠ 0)
  • a^(1/n) = √[n]{a}

Логарифми

  • logₐ(b·c) = logₐb + logₐc
  • logₐ(b/c) = logₐb − logₐc
  • logₐ(bⁿ) = n·logₐb
  • logₐa = 1,   logₐ1 = 0

Арифметична прогресія

aₙ = a₁ + (n − 1)d

Sₙ = (a₁ + aₙ)/2 · n

Геометрична прогресія

bₙ = b₁·q^(n−1) (q ≠ 1)

Sₙ = b₁·(qⁿ − 1)/(q − 1)

Комбінаторика

Pₙᵏ = n! / (n−k)! ,   Cₙᵏ = n! / (k!·(n−k)!)

Теорія ймовірностей

P(A) = k/n

Геометрія

Елементарні фігури

  • Сума внутрішніх кутів n-кутника: (n − 2)·180°
  • Кількість діагоналей: n(n − 3)/2

Трикутник

S = ½·a·b·sin γ

a² = b² + c² − 2bc·cos α (теорема косинусів)

a/sin α = b/sin β = c/sin γ = 2R (закон синусів)

Паралелограм

S = a·hₐ

Ромб та трапеція

Ромб: S = (d₁·d₂)/2

Трапеція: S = ((a + b)/2)·h

Коло і круг

C = 2πr,   S = πr²

Сектор: S = πr²·(φ/360°),   Дуга: l = 2πr·(φ/360°)

Циліндр, конус, сфера

Циліндр: V = πr²h,   Sₚₒₗₙ = 2πr(h + r)

Конус: V = (1/3)πr²h,   Sₚₒₗₙ = πr(l + r)

Сфера: V = (4/3)πr³,   S = 4πr²

Координати та вектори

  • Відстань AB: √((x₂−x₁)²+(y₂−y₁)²+(z₂−z₁)²)
  • Скалярний добуток: u·v = u₁v₁ + u₂v₂ + u₃v₃
  • cos φ = (u·v)/(‖u‖·‖v‖)
